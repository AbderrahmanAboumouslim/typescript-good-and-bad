### Installation

```sh
npm install typescript -g ts-node
```

### Compile

```sh
tsc index.ts
```

### Run

```sh
node index.js
```

### Combine the commands Compile & Run

```sh
ts-node index.ts
```

## Fix Run error

In case your you find an error in the run command, try:

```sh
npm install @types/node --save-dev
```
